NeuroScan — AI EEG Seizure Detection

An educational demo combining classical EEG signal processing with machine learning.

What it does

NeuroScan ingests raw EEG recordings in the standard .edf format, applies bandpass filtering and feature extraction, and runs a lightweight classifier to flag windows that look like ictal (seizure) activity.

Dataset

Designed to work with the CHB-MIT Scalp EEG Database (specifically chb01 recordings) hosted on PhysioNet. The CHB-MIT dataset contains EEG recordings from pediatric subjects with intractable seizures — the gold standard for benchmarking seizure detection models.

The Problem

Manual review of long-duration EEG recordings is time-consuming and demands expert neurologists. A patient might be monitored for days; a clinician must scan every channel for telltale spikes, sharp waves, and rhythmic discharges.

The Solution

An automated pipeline that:

  • Loads multi-channel EEG via MNE / pyEDFlib
  • Bandpass filters (0.5–40 Hz) and z-score normalizes each channel
  • Segments the signal into 5-second windows
  • Extracts statistical (mean, variance, energy, entropy) and spectral features (delta, theta, alpha, beta, gamma band powers)
  • Predicts seizure probability per window using a Random Forest
  • Flags abnormal regions and visualizes waveform, probabilities, and band powers

Future Scope

  • Train on the full CHB-MIT corpus (chb01–chb24) with patient-aware cross-validation
  • Replace Random Forest with a 1D-CNN or transformer trained end-to-end on raw windows
  • Real-time streaming inference for bedside monitoring
  • Explainability via SHAP and signal-level saliency maps
  • Multi-class classification (focal vs generalized, pre-ictal warnings)
